## Why Puppeteer stealth plugins fail against DataDome in 2024

DataDome isn't checking your user-agent string and calling it a day. It's running a full environment fingerprint:

- TLS/JA3 fingerprinting — your headless Chrome's TLS handshake looks nothing like a real browser's
- Canvas and WebGL hash comparison
- JavaScript execution timing analysis
- Navigator property enumeration (the `webdriver` flag is just the start)
- HTTP/2 frame ordering and header priority signals

Puppeteer-extra-plugin-stealth patches maybe40% of these vectors. It spofs `navigator.webdriver`, fakes plugins, overides `chrome.runtime`. But DataDome's detection stack operates layers below what a browser-level plugin can reach.
